Pixem

An intuitive unit conversion tool for stylesheets

Pixem provides developers with the means to convert pixel units to the equivalient value in ems or rems.

Start Using Pixem

How to Use Pixem

Paste in your stylesheet

The style sheet should contain pixel values to be converted to either REMs or EMs.

Set base pixel size

This should be the top-level font-size on your webpage. The default for most web browsers is 16 pixels.

Preserve original values

Check whether to preserve original values with CSS comments. This simply means that the original pixel value before the conversion is preserved in a comment next to the converted value.

Choose conversion unit

Choose a unit to convert pixel values to—em or rem. The former is relative to the font-size of the direct parent and the latter is relative to the font-size of the root element.

Specify the affected properties

Indicate which CSS properties should be converted by typing in a list of comma-separated properties (e.g., “font, font-size”). If the field is empty, all properties will be affected by the conversion.

Copy the converted stylesheet

Copy the converted stylesheet to your clip board. The result can be found in the text area that is right next to the input.
